```html
<!--修改 display 属性 flex 对比-->
<html>
<head>
    <meta charset="UTF-8">
    <title>Display PropertyTest</title>
    <style type="text/css">
        .test-display
        {
            display: flex;
            display: -webkit-flex;
            /*要支持 Safari，必须添加 -webkit- 前缀：*/
        }
    </style>
</head>
<body>
    <div class="test-display">
        <div> zhangyuwei hello world!</div>
        <div> This is a big company!</div>
    </div>
</body>
</html>
```
